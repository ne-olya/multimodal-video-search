# Инструкция по оценке релевантности

Оценка ставится содержанию видео, а не названию файла или популярности автора.

- `0` — содержание не соответствует запросу;
- `1` — присутствует похожий объект или контекст, но нет нужного действия;
- `2` — запрос в основном выполнен, есть небольшая неоднозначность;
- `3` — точное соответствие с наблюдаемым временным фрагментом.

Для оценок `2–3` annotator указывает `evidence_start/end` и основную модальность. Не показывайте annotator similarity score модели. Для 10% пар нужна двойная разметка; согласие измеряется weighted Cohen's kappa.
